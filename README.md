# SHOPPING-LIST.Java
Name: Intertas Kate winzlet T.

Program Title: Shopping List Project

Description: This program allows the user to create a simple shopping list. The user can input up to 5 items they want to buy. If no item is entered, the program stops asking for more input and displays all the items that were added to the list. It helps the user organize what they plan to purchase in a simple way. Sample Input and Output:

Enter up to 5 items to buy: Milk Bread Eggs Butter Cheese

Your shopping list: Milk, Bread, Eggs, Butter, Cheese

You entered 5 items.

Search for an item: Eggs Eggs is in your shopping list.
